# Rassvet — AI Library for Psychologists

**rassvet-ai.com**

Rassvet is an AI-powered semantic library for practicing psychologists, 
psychiatrists, and neuroscientists. It searches professional psychology 
literature and returns answers grounded strictly in published books — 
with author, title, and page citation for every claim. No hallucinations.

## What it does

- Semantic search across a growing library of psychology, psychiatry, 
  and neuroscience books (CBT, ACT, DBT, schema therapy, psychoanalysis)
- Every answer cites the exact source: author, book, page
- Free tier: 3 questions, no signup required
- Available in Russian, English, and Arabic

## Tech stack

Next.js · FastAPI · FAISS + SentenceTransformer + CrossEncoder reranking · 
GPT · SQLite · Redis
